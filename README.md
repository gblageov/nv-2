# NikiVibes Image Processor

Програма за обработка на Excel файлове с продукти и техните снимки.

## Инсталация

1. Инсталирайте Python 3.8 или по-нова версия
2. Клонирайте репозиторито
3. Създайте виртуална среда:
   ```
   python -m venv venv
   ```
4. Активирайте виртуалната среда:
   - Windows: `venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`
5. Инсталирайте зависимостите:
   ```
   pip install -r requirements.txt
   ```
6. Копирайте .env.example във .env и попълнете необходимите настройки

## Употреба

1. Поставете вашия Excel файл в папката `data/`
2. Изпълнете `start.bat` (Windows) или `python main.py` (други ОС)
3. След приключване на обработката, ще бъдат генерирани:
   - Нов Excel файл с добавете снимки
   - Лог файл с информация за извършените действия

## Файлова структура

- `main.py` - Основен скрипт за обработка
- `requirements.txt` - Зависимости на проекта
- `.env` - Конфигурационни настройки
- `data/` - Папка за входни и изходни файлове
- `logs/` - Папка за лог файлове

## Лог файл

След всяко изпълнение се генерира лог файл с информация за:
- Успешно обработени продукти
- Липсващи ID-та от външния файл
- Грешки по време на изпълнение
